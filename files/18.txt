AWS is a powerful cloud service provider from Amazon, one of the giants in the ECommerce market. With the Command Line Interface tool, AWS CLI enables users to control all the AWS resources using the command in the Power Shell. Some of the advantages of AWS are: 
- Saving time: Instead of downloading the countless number of packages and subset files, AWS saves users much time by supporting all the services with only one tool.  - Scripting: with the control over the resource, The Command Line Interface Power Shell automate all the services, developers do not need to always stay on the station to keep track or control what is happening which saves a great deal of time, workforce, and increase productivity in different tasks.
